Port Website

GoodJob เป็นเว็บไซต์ที่รวบรวมผลงานของสมาชิกแต่ละคนในกลุ่ม จัดทำผ่านภาษา HTML CSS และ Javascript ซึ่งเป็นงานโปรเจคของรายวิชา Web Development I (CMM240) โดยสมาชิกแต่ละคนจะนำผลงานด้านต่างๆของตนเอง มารวบรวมไว้ในเว็บไซต์ ไม่ว่าจะเป็นผลงานด้านการ Design, Drawing, Video รวมถึงผลงานด้านโปรแกรมต่างๆ ทั้งผลงานที่ทำผ่านโปรแกรม Adobe Photoshop, Adobe Indesign, Adobe Illustrator, Adobe After Effects และโปรแกรมอื่นๆอีกมากมาย โดยแต่ละผลงานล้วนมาจากฝีมือและประสบการณ์ของสมาชิกแต่ละคนทั้งหมด
